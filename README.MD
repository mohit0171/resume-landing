🚀 ResumeFlow —
The AI-Powered Resume Builder Landing PageA hyper-clean, dark-themed, conversion-focused landing page for ResumeFlow—a next-generation AI resume builder. This project is a masterclass in modern semantic layouts, proving that you don't need massive frameworks or endless nested layouts to build a stunning, interactive user experience. 


✨ The Philosophy:
Zero Bloat, Ultimate PerformanceMost modern landing pages are buried under a mountain of heavy javascript framework code. ResumeFlow takes a strict structural approach: No Framework Bloat: Fast-loading vanilla stack.Semantic Integrity: Built exclusively with semantic tags. Pure Layout Mastery: Zero reliance on unsemantic structural tag wrappers to control layouts or grid positioning.

🛠️ Feature Highlights

⚡ Smart Accordion (FAQ Section)
The FAQ grid uses native <details> and <summary> elements styled with advanced CSS grid transitions.Auto-Close Synergy: Opening a new question automatically collapses the currently active one smoothly.Dynamic Micro-Copy: Powered by script.js, the subheader content dynamically tallies the total available questions using querySelector and textContent automatically.The "+" to "×" Transformation: A pseudo-element icon at the end of each item that rotates and changes color dynamically based on the toggle state.


🎨 Design
SystemBuilt on a dark color scheme with fluid typography and smooth hover interactions: Glassmorphism Header: A sticky navigation bar utilizing a backdrop-filter blur effect. Gradient Textures: Subtle linear-gradient text properties to give the typography an ultra-premium aesthetic. Bento-Style Grids: Dynamic feature cards and template previews built entirely on CSS Grid.
